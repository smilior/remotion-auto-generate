# ナレーション原稿: Claude Code ショートカットキー 5選
# 想定尺: 約30秒（約120文字）

## シーン構成

### [0-3秒] タイトル
（ナレーションなし — タイトル表示のみ）

### [3-25秒] 本編

Claude Codeを使いこなすなら、ショートカットキーは必須です。

まず、Ctrl+Cで応答をすぐ止められます。
Ctrl+Lで画面をスッキリ。
Escapeで入力をやり直し。
スラッシュを打てばコマンド一覧が出ます。
そしてTabで補完。これだけで操作が格段に速くなります。

### [25-30秒] エンディング
（ナレーションなし — CTA表示のみ）

---

## ターミナル表示用コマンド

```
$ claude --shortcuts

1  Ctrl + C   — 応答をキャンセル
2  Ctrl + L   — 画面をクリア
3  Escape     — 入力をキャンセル
4  /command   — スラッシュコマンド
5  Tab        — オートコンプリート
```
